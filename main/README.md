# Paradise Nursery

Paradise Nursery is a React plant shop app built with Redux Toolkit and React Router.

## Pages
- Landing page with a background image and a `Comenzar` button
- About Us page with a simple English description
- Products page
- Shopping cart page

## Files requested by the rubric
- `README.md`
- `src/AboutUs.jsx`
- `src/App.css`
- `src/App.jsx`
- `src/CartSlice.jsx`
- `src/ProductList.jsx`
- `src/CartItem.jsx`
